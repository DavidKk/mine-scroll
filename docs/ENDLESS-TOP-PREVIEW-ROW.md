# 无尽模式 · 顶行半隐藏预览带

> 单一事实来源：实现顶行预览时以本文为准，勿自行加「假数字 / 常驻蒙层 / 禁止点击」等未写明行为。

## 是什么

棋盘**可玩区正上方**多露出 **上一行（`viewStart - 1`）的下半截**，像俄罗斯方块看到「下一块」进场的方向感，但**仍是真实棋盘格**，不是装饰条。

## 视觉

| 项   | 要求                                                                      |
| ---- | ------------------------------------------------------------------------- |
| 高度 | `ENDLESS_PREVIEW_ROWS`（当前 **0.5 行**）                                 |
| 内容 | 与可玩区**同一套** `drawCell`：未开=盖格，已开=数字/雷/旗                 |
| 半隐 | **渐变淡出**（上透明 → 下清晰），**禁止**硬裁剪一刀黑边                   |
| 布局 | 棋盘 Canvas 顶部永久预留 0.5 行带宽；`viewStart === 0` 时无上一行则不画格 |

## 交互（重要）

| 项           | 要求                                                                    |
| ------------ | ----------------------------------------------------------------------- |
| 可点击       | **可以点**。点在可见（渐变下半）区域内，等同操作 `viewStart - 1` 那一行 |
| 开格         | 左键翻开 → 正常 flood fill → **显示真实数字**                           |
| 插旗 / Chord | 与可玩区规则相同                                                        |
| 命中         | 屏幕行 **-1** 映射 `endlessScreenRowToLocal(session, -1)`               |
| 不可点       | 渐变已淡到看不见的上半区域（可选，实现上仅命中可见半行）                |

## 不是什么

- ❌ 不是一直用半透明假 `revealed` 偷看数字
- ❌ 不是「顶行没点开就不显示预览带」
- ❌ 不是不可点的纯装饰
- ❌ 不是硬 `clip` 后盖黑色蒙层

## 与卷轴的关系

- 预览行是缓冲里的**真实上一行**；卷轴上移后该行进入 `viewStart`，变为普通第 0 行。
- 已翻开格子的动效**不因卷轴重播**（用 `fxKey` 世界坐标去重）。

## 代码落点

| 模块              | 职责                                                                      |
| ----------------- | ------------------------------------------------------------------------- |
| `endless.ts`      | `toEndlessCellViews` 输出 `preview: true` 行；`actionableBounds` 含预览行 |
| `renderer.ts`     | 渐变半隐绘制；`hitTestCellWithPreview`                                    |
| `game-canvas.ts`  | 命中 -1 行；`cellAtCoords` 放行                                           |
| `game-session.ts` | `isEndlessInteractiveScreenRow` 含 `-1`                                   |

## 验收

1. 开局顶缘可见半行盖格，渐变自然，无黑条。
2. 点击可见半行可翻开，数字与可玩区一致。
3. 插旗/Chord 在预览行可用。
4. 卷轴上移后不重复星星/爆炸动效。
