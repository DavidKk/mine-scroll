| ID            | 名称       | 核心差异                                       | 状态 |
| ------------- | ---------- | ---------------------------------------------- | ---- |
| `classic`     | 经典扫雷   | 方形 8 邻格，可配置难度                        | ✅   |
| `hex`         | 六边形扫雷 | 正六边形盘，6 邻格，固定 91 格 / 10 雷         | ✅   |

## 各模式说明

### classic — 经典扫雷

与 `SPEC.md` 一致。含难度配置、Chord、首击安全。

### hex — 六边形扫雷

- **正六边形盘**（半径 5 → 91 格），固定 **10 雷**
- 每格 **6 邻**（非 8 邻），数字 0–6
- 经典规则：开格 / 插旗 / Chord / 首击安全（中心格 + 6 邻）
- 六边形 Canvas 渲染与独立 hit-test

### endless — 无尽卷轴

- 固定 **9 列**，屏幕可见 **11 行**（内部 14 行缓冲）
- **雷 ∞**：按 `(世界行, 列, 种子)` 生成；密度 **21% → 30%** 随卷轴升高渐增
- **首击安全**（点击格 + 8 邻）
- **卷轴**：双轨阶梯 **严格交替**——每 **50s** 升一档，偶数步加快倒数（9.0s→1.5s 共 11 档）、奇数步加批量（1→5 行/次）；**每次事件最多 −1 命**（多行离屏不叠扣）；顶栏 `↑04×3` 显示倒数与批量；**「上移」/ AI 上移**跟当前批量档；手动上移重置倒数
- **连通展开**：顶行新格若与已翻开 **0 区**相连，自动 flood，减少纯猜
- **底行扣血**：离屏底行只要有任意未处置格 **−1 生命**（每轮最多扣 1）；**已踩雷扣过命**或**已翻开**的格不再重复扣；**正确插旗的雷**不算漏格；底行全空白免扣
- **踩雷**：点雷或 Chord 踩雷 **−1 命/颗**，生命耗尽才失败
- 共 **5 命**；无胜利条件
- **消雷 / 回血**：底行上移离屏时，**正确插旗的雷**入账；**4 消雷 = 1 命**（手动「回血」，上限 5 命）；同一雷只计一次
- HUD：**生命值** + 消雷数 + 卷轴高度；支持插旗 / Chord
- **AI 助手**：子集推理 + CSP 概率猜雷；底行漏格时锁定上层；临期连走；无尽禁止盲猜；**消雷 ≥4 且未满命时自动回血**；**底行零漏格且推理确认时手动上移加速**

### puzzle-rush — 连击清盘

> 动态难度详见 **`docs/PUZZLE-RUSH-DIFFICULTY-PLAN.md`**（草案）。以下为当前已实现 + 方案摘要。

- 路由 **`/play/rush`**；ranked 验证；独立排行榜（`mode=puzzle-rush`）
- 固定 **7×7**；开格 / 插旗 / Chord / 首击安全
- **3 命**；踩雷 −1 命、连击归零；连续 **4 盘**无扣血 **+1 命**（上限 3）
- **清盘换局**：滑入过渡（约 0.68s），期间锁定输入
- **得分（方案）**：每盘 `100 + 速通奖励`；连击 **不计分**，仅影响难度
- **难度（v0.6）**：
  - **新手局**：前 3 盘固定 5→6→7 雷、纯技术；清完第 3 盘 → **RAMP UP** 提示进入热身
  - 三轨道：雷密度（只升不降）/ 盘面类型（纯技术 vs 掺杂）/ 会话掺杂配额
  - 阶段 **双轴取慢**：`min(有效时长, 盘数进度)`，每档至少 5 盘；有效时长 = `min(墙钟, 进度盘数×60s)`
  - 连击 ≥10 / 压力 ≥60 升掺杂（非 intro/热身）；下一盘首击生效；无硬结束
  - 保命：M0 → M1 → M2 → M3（apex）

**雷密度档**（intro 3 盘不计入进度）：

| 进度盘序 | 雷数 | 速通窗口 |
| -------- | ---- | -------- |
| 1–5      | 8    | 60s      |
| 6–10     | 8    | 55s      |
| 11–15    | 9    | 50s      |
| 16–20    | 9    | 45s      |
| 21–25    | 10   | 40s      |
| 26–30    | 10   | 38s      |
| 31+      | 11   | 35s→33s  |
