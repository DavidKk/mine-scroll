# MineScroll

浏览器里就能玩的**在线扫雷**——霓虹棋盘、排位对局、实时排行榜。

**立即游玩 → [mine-scroll.vercel.app](https://mine-scroll.vercel.app)**

无需安装。点开链接，选模式，开始排雷。

## 怎么玩

经典扫雷规则：

- **开格** — 左键 / 点击未翻开的格子
- **插旗** — 右键 / 滑动标记地雷
- **Chord** — 双击 / 双击数字，旗数对齐时一次开完邻格

## 玩法模式

| 入口                                                 | 模式    | 你要做什么                           |
| ---------------------------------------------------- | ------- | ------------------------------------ |
| [首页](https://mine-scroll.vercel.app)               | Landing | 了解玩法、看公开排行榜               |
| [无尽卷轴](https://mine-scroll.vercel.app/play)      | Endless | 棋盘向上滚——5 命撑住卷轴，叠连击冲榜 |
| [连击清盘](https://mine-scroll.vercel.app/play/rush) | Rush    | 连续清 7×7，叠倍率冲 rush 榜         |

## 为什么玩 MineScroll

- Endless / Puzzle Rush 排位榜，分数可校验
- 桌面和手机都能玩
- 扫雷手感不变，街机表现更爽
