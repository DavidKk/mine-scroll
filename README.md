# 扫雷 Web 游戏

经典 Windows 风格扫雷 MVP（9×9 / 10 雷）。

包管理器：**pnpm**（见 `package.json` 的 `packageManager` 字段）。

## 开发

```bash
pnpm install
pnpm dev
```

## 文档

- `docs/PROJECT.md` — 迭代进度与 Current Task
- `docs/SPEC.md` — 游戏规则
- `docs/ARCHITECTURE.md` — 技术架构

## 构建

```bash
pnpm build
pnpm preview
pnpm test
```
